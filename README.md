# EventEase

A minimalist event-planning website for **Nexus**.

## Pages

- `faq.html` — Frequently Asked Questions help centre with live search, category filters, and an accordion. Styles live in `css/faq.css`; interactivity in `js/faq.js`.

## Running locally

No build step required — it's a static site. Open `faq.html` directly in a browser, or serve the folder:

```bash
python -m http.server 8000
# then visit http://localhost:8000/faq.html
```
